AutoName: AlleyCat GPG

Summary: Encrypt and decrypt messages and files for secure SMS communication

Description: |
  AlleyCat GPG lets you encrypt and decrypt text messages and files using
  OpenPGP public-key cryptography — without needing the command line.

  The intended use case is sending encrypted content over any channel that
  carries text or files: SMS, email, Signal, Telegram, etc. Your carrier
  and ISP see only opaque ciphertext.

  Features:
  * Compose and encrypt messages to PGP key recipients
  * Passphrase-only (symmetric) encryption — no PGP key required
  * Sign + encrypt with your own key
  * Decrypt armored PGP messages with signature status display
  * Generate Ed25519 or RSA 4096 keypairs on-device
  * Import contacts' public keys by paste or QR code scan
  * Export your public key as QR code for easy sharing
  * File encryption for any file type (PDF, image, zip, etc.)
  * Key fingerprint verification with out-of-band verification reminder

  Security design:
  * All cryptography via the open-source openpgp Dart library
  * Private keys stored in Android's encrypted KeyStore
  * No plaintext is stored, logged, or transmitted
  * No network requests — fully local operation
  * No telemetry or analytics

  This app is designed to complement Signal and similar tools, not replace
  them. It covers scenarios where you need to encrypt content delivered over
  channels you don't control (SMS, email) and want to avoid ISP or carrier
  visibility into your message content.
